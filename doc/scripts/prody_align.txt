Usage: prody align [options] PDB  

Align models in PDB file using selected atoms and save aligned coordinate sets.

Options:
  -h, --help            show this help message and exit
  --examples            show usage examples and exit
  --quiet               don't print log messages to stderr
  -p STRING, --prefix=STRING
                        prefix for output files, default is PDB_aligned
  -s STRING, --select=STRING
                        selection string, default is "calpha"
  -m INT, --model=INT   model index onto which other models will be
                        superposed, default is 1
