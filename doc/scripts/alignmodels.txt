Usage: ../scripts/alignmodels.py [options] PDB  

ProDy v0.6 - Align Models

Align models in the PDB file using selected atoms and save aligned coordinate
sets.



Options:
  -h, --help            show this help message and exit
  --silent              omit verbose information, default is False
  -p STRING, --prefix=STRING
                        prefix for output files, default is PDB_aligned
  -s STRING, --select=STRING
                        selection string, default is "calpha"
  -m INT, --model=INT   model index onto which other models will be
                        superposed, default is 1
