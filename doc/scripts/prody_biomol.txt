Usage: prody biomol [options] PDB  

Generate biomolecule coordinates.

Options:
  -h, --help            show this help message and exit
  --examples            show usage examples and exit
  --quiet               don't print log messages to stderr
  -p STRING, --prefix=STRING
                        prefix for output files, default is PDB_biomol_
  -b INT, --biomol=INT  index of the biomolecule, default is "none"
