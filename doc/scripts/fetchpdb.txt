Usage: ../scripts/fetchpdb.py [options] PDB PDB2 PDB3 ...  

ProDy v0.6 - Fetch PDB

Download PDB files specified by their identifiers.


Options:
  -h, --help            show this help message and exit
  --examples            show usage examples and exit
  --quiet               don't print status messages to stdout
  --version             print ProDy version and exit
  -d PATH, --dir=PATH   target directory saving downloaded PDB files
  -f FILE, --file=FILE  file that contains PDB identifiers
