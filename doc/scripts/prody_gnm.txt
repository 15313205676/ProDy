Usage: prody gnm [options] PDB

Perform GNM calculations for given PDB structure and output results in NMD 
format. If an identifier is passed, file from the PDB FTP server will be 
downloaded.

Options:
  -h, --help            show this help message and exit
  --examples            show usage examples and exit
  --quiet               don't print log messages to stderr

  Parameters:
    -c FLOAT, --cutoff=FLOAT
                        cutoff distance, default is 10.0 A
    -g FLOAT, --gamma=FLOAT
                        spring constant, default is 1.0
    -m INT, --model=INT
                        model that will be used in the calculations
    -n INT, --number-of-modes=INT
                        number of non-zero eigenvalues/vectors to calculate,
                        default is 10
    -s STRING, --select=STRING
                        selection string, default is "calpha"

  Output:
    -a, --all-output    write all outputs
    -o PATH, --output-dir=PATH
                        output directory, default is "."
    -e, --eigenvs       write eigenvectors/values
    -r, --cross-correlations
                        write cross-correlations
    -q, --square-fluctuations
                        write square-fluctuations
    -v, --covariance    write covariance matrix
    -z, --npz           write compressed ProDy data file
    -b, --beta-factors  write B-factors
    -k, --kirchhoff     write Kirchhoff matrix

  Output Options:
    -p STRING, --file-prefix=STRING
                        prefix for output files, default is "PDB_gnm"
    -f STRING, --number-format=STRING
                        delimiter, default is "%12g"
    -d STRING, --delimiter=STRING
                        delimiter, default is " "
    -x STRING, --extension=STRING
                        file extension, default is ".txt"

  Figures:
    -A, --all-figures   save all figures
    -R, --cross-correlations-figure
                        save cross-correlations
    -Q, --square-fluctuations-figure
                        save square-fluctuations
    -B, --beta-factors-figure
                        save beta-factors
    -K, --contact-map   save contact map (Kirchhoff matrix)
    -M STRING, --mode-shape-figure=STRING
                        save mode shape figures for specified modes, e.g. "1-3
                        5" for modes 1, 2, 3 and 5

  Figure Options:
    -F STRING, --figure-format=STRING
                        figure format, one of eps, pdf, png, ps, raw, rgba,
                        svg, svgz, default is "pdf"
    -D INT, --resolution=INT
                        figure resolution (dpi), default is 300
    -W FLOAT, --width=FLOAT
                        figure width (inch), default is 8.0
    -H FLOAT, --height=FLOAT
                        figure height (inch), default is 6.0
