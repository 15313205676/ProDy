Usage: ../scripts/blastpdb.py [options] SEQUENCE
    
ProDy v0.8.1 - Blast PDB

SEQUENCE can be a sequence string or a file in fasta format.  


Options:
  -h, --help            show this help message and exit
  --examples            show usage examples and exit
  --quiet               don't print status messages to stdout
  --version             print ProDy version and exit
  -c FLOAT, --coverage=FLOAT
                        percent coverage, default is 90.0%
  -d PATH, --dir=PATH   if given, download PDB files to the folder
  -i FLOAT, --identity=FLOAT
                        percent sequence identity, default is 90.0%
